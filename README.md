# 📈 Stock Market Intelligence Pipeline

A production-ready ETL pipeline for stock market data with anomaly detection and real-time dashboard.

## 🚀 Features

- Fetches real-time stock data from Alpha Vantage API
- Detects anomalies using Isolation Forest (Machine Learning)
- Interactive dashboard with Streamlit
- Data quality checks and validation
- Async API calls for fast performance

## 🛠️ Tech Stack

- Python 3.11+
- Streamlit (Dashboard)
- Pandas, NumPy (Data Processing)
- scikit-learn (Anomaly Detection)
- SQLite (Database)
